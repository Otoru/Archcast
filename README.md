# Wireframe

Projeto Next.js para wireframes, com Tailwind CSS, TypeScript, Vitest, Base UI (variante shadcn) e Storybook.

## Scripts

```bash
pnpm dev            # dev server
pnpm build          # build de produção
pnpm lint:fix       # biome check --write
pnpm test           # vitest (watch)
pnpm test:run       # vitest run (CI)
pnpm storybook      # storybook dev em :6006
pnpm build-storybook
```
